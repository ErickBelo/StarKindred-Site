# Documentação visual da StarKindred

Esta pasta será usada para registrar decisões, ideias e alterações relacionadas ao frontend do site StarKindred.

## Diretrizes

- Preservar os `id`, `class` e atributos usados pelo JavaScript e pelo Firebase.
- Priorizar alterações visuais em CSS e reutilização da estrutura HTML existente.
- Manter a identidade visual espacial, competitiva e comunitária da StarKindred.
