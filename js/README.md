# Organização de JavaScript

Esta pasta será usada para separar comportamentos por página.

- `index.js`: interações visuais da homepage, como o painel de elite e animações auxiliares.
- `login.js`: login e alistamento.
- `painel.js`: funcionalidades da cabine de comando.
- `admin.js`: funcionalidades administrativas.

Não mover ou renomear scripts críticos sem verificar as referências existentes. A lógica de banco de dados deve continuar sendo tratada separadamente da camada visual.
